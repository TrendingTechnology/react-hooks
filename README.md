# 📦 Install

```
$ npm i @vortigo/react-hooks
or
$ yarn add @vortigo/react-hooks
```

# 🔨 Usage

```ts
import { useMount } from '@vortigo/react-hooks';
```

# 💻 How to run the example pages

```
npm run parcel <path-to-example-page>
```
